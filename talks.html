Combinatorial aspects of surface cluster Algebras: An introduction

On Feb 24th 23, I gave a quick introduction to my bachelor thesis; more precisely, combinatorial aspects of surface Cluster algebras and applications to Frobenius' conjecture. The main part of my talk was focused on explaining the process of generating a Cluster algebra by using the triangulated puncture torus to construct a cluster quiver to then define a cluster mutation that generates all Markov triples.
